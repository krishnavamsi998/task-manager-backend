# Task Manager Backend

A backend API for a task management application built with **Node.js**, **Express**, and **TypeScript**.

## Features

- RESTful API for managing tasks
- Create, read, update, and delete tasks
- Structured with controllers, routes, and models

## Tech Stack

- **Node.js**
- **Express**
- **TypeScript**
- **MongoDB** (configured via env)
- **Redis** (configured via env)

## To spin up server:
1. git clone https://github.com/krishnavamsi998/task-manager-backend.git
2. cd task-manager-backend
3. npm install
4. npm run dev
result: starts local dev server for backend.

Note: please ensure you have MongoDB and Redis installed and enabled
